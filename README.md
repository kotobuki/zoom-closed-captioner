# Zoom用簡易字幕サンプル - Simple captioner for Zoom example

Web Speech APIで音声認識したテキストをZoomの字幕として表示するサンプルです。

あくまでサンプルコードで、かつ内容も整理中であり、一般公開を想定したものではありません。今後の予定についてはTODO.mdをご覧ください。
